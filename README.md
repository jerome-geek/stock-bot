# Stock Analysis Bot (3x Leverage ETFs)

이 프로젝트는 3배 레버리지 ETF(SOXL, TQQQ 등)의 변동성을 통계적으로 분석하고, 과매도 구간을 포착하여 구글 스프레드시트에 자동으로 기록하는 봇입니다.

## 📈 주요 기능

-   **표준편차 분석**: 252일(1년 영업일) 기준 이동평균 및 표준편차 계산.
-   **Z-Score 산출**: 현재 주가가 통계적으로 어느 위치에 있는지 분석.
-   **매수 신호 알림**: -1σ, -2σ, -3σ 이탈 시 단계별 매수 신호 발생.
-   **자동 업데이트**: Google Sheets API 연동 및 GitHub Actions를 통한 일일 자동 실행.

## 🚀 시작하기

### 1. 환경 설정

이 프로젝트는 [uv](https://astral.sh/uv/)를 사용합니다.

```bash
# 의존성 설치 및 가상환경 설정
uv sync
```

### 2. GCP 및 구글 시트 세팅

분석 결과를 저장할 구글 스프레드시트와 API 설정이 필요합니다.
자세한 내용은 [GCP_SETUP_GUIDE.md](./GCP_SETUP_GUIDE.md)를 참고해 주세요.

### 3. 로컬 실행

```bash
uv run main.py
```

## 📂 프로젝트 구조

-   `main.py`: 프로그램 실행 진입점.
-   `src/`: 데이터 수집, 분석, 시트 연동 핵심 로직.
-   `config/`: 티커 설정 및 인증 정보 로더.
-   `.github/workflows/`: GitHub Actions 일일 자동화 설정.

## ⚖️ 면책 조항

본 프로그램은 정보 제공만을 목적으로 하며, 투자 권유를 목적으로 하지 않습니다. 모든 투자 결정의 책임은 투자자 본인에게 있습니다.
